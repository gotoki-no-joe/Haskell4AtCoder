# 18.最大公約数、最小公倍数

HaskellではPreludeにgcdとlcmがある。

3つ以上の値のそれも、繰り返し適用すれば求められる。

```haskell
gcdList, lcmList :: [Int] -> Int
gcdList = foldl1' gcd
lcmList = foldl1' lcm
```
